# Tokusho 紹介動画 スクリプト（現行UIに対応・改訂版）

> ナレーション音声は `voiceover/scene1〜5.mp3`（変更不要・そのまま使える）。
> 収録前に: ①最新テンプレで**再公開**しておく（タイトル重複なし等） ②`事業者情報`の「再取込」+保存で**テストデータ(テスト商店)を実データ化** ③ウィンドウ幅を広めにして表が2カラム表示になる状態で録画。

| 凡例 | 意味 |
|---|---|
| 🎙 | ナレーション（既存MP3） |
| 🖥 | 画面・操作（現行UIの実際のラベル） |
| 🔻 | 旧台本からの変更点 |

## シーン1 — 導入（0:00–0:20）
🖥 Shopify 管理画面 → 左サイドの **Tokusho** を開く → ダッシュボード「特定商取引法ページ管理」。
🎙「Tokushoは、特定商取引法に基づく表記ページを、入力するだけで簡単に作成・公開できるアプリです。本日は、導入から公開までの流れをご紹介します。」
🔻 変更なし。

## シーン2 — かんたん設定 / 自動入力（0:20–1:00）
🖥 サイドの **事業者情報** → 上部 **かんたん設定** で販売形態（例：**物理商品（個人事業主）**）を選ぶ → 関連項目が自動セット・不要欄（送料等）が省かれる様子を見せる → **「Shopify設定から取込む」**ボタンを押し、事業者名・住所・電話・メールが自動入力される様子。
🎙「まず、販売形態を選びます。選ぶだけで関連項目が自動でセットされ、不要な欄は省かれます。Shopifyの設定から、事業者名や住所も自動で取り込めます。」
🔻 **商品の自動判定（デジタル/サブスクの提案バナー）は廃止**したので映さない。「販売形態を選ぶ」＋「Shopify設定から取込む（自動入力）」を見せればナレーションと一致。

## シーン3 — 詳細入力（1:00–2:10）
🖥 下にスクロールし **3. 販売条件**（販売価格・お支払い方法・**お支払い時期/商品の引渡し時期はプルダウンから選択**）→ **4. 返品・キャンセルポリシー**（返品ポリシーを選択肢から）。空欄には**「例：…」のヒント**が出る点も自然に見せる。
🎙「必要な項目を順番に入力します。販売価格や支払い方法、商品の引渡し時期、そして特商法で省略できない返品ポリシーも、選択肢から簡単に設定できます。」
🔻 引渡し時期・支払い時期・返品条件などは**プルダウン＋「自由に入力する」**に変更済み（ナレーションの「選択肢から」とむしろ合致）。デザイン（アクセントカラー＝色を選ぶUI）はProのため、Free説明では省略可。

## シーン4 — プレビュー・公開（2:10–3:00）
🖥 サイドの **プレビュー・公開** → 生成された「特定商取引法に基づく表記」表（**ウォーターマークなし**、項目名は**日英併記**、都道府県は漢字）→ **「ページを更新して公開する」**を押す → 緑の「公開しました！」バナー＋URL＋「ページを表示」。
🎙「内容を確認したら、プレビュー画面で仕上がりをチェックします。問題がなければ、ワンクリックで公開。お客様のストアに、表記ページが自動で作成されます。」
🔻 Freeでも**ウォーターマークなし**／**日英併記**が映る点が以前より良い見せ場。「ページを表示」で実際のストアページをチラ見せすると説得力アップ。

## シーン5 — 追加機能・まとめ（3:00–3:40）
🖥 サイド **プライバシーポリシー**（Pro・フォーム）→ **追加ページ**（会社概要/お問い合わせ/返品ポリシー）を素早く見せる。可能ならダッシュボードの **「全ページをワンクリックで生成」(Pro)** で5ページ一括プレビュー→公開も一瞬見せる。
🎙「プライバシーポリシーや、追加ページの作成にも対応しています。Tokushoなら、面倒な法令表記の準備を、誰でも数分で完了できます。」
🔻 **新セールスポイント**: ①「全ページをワンクリックで生成（5ページ）」②法令変更時の**お知らせ**（ダッシュボードのバナー）。映すのは任意（ナレーションは現行のままでOK）。

## 任意で足したい新機能（ナレーション追加＝MP3再生成が必要）
現行MP3はシーン1〜5そのまま使えます。下記を強調したい場合のみ追記:
- **「Freeでもウォーターマークなし」**（無料の強さ）
- **「5ページをワンクリックで一括生成」**（Proの価値）
- **「法令変更があればお知らせ」**（継続利用の価値）
→ 例の追加ナレーション例:「無料プランでもウォーターマークは付きません。Proなら、プライバシーポリシーや会社概要など5ページをワンクリックで一括作成。法令の更新があればお知らせします。」
（必要なら `voiceover/generate_voiceover.py` に scene6 を足して再生成可能。）

## 収録メモ
- ツール: Windows の **Snipping Tool（録画）** で範囲指定、または Xbox Game Bar（Win+Alt+R）。1600×900 目安。
- 音声は無音で録画 → 後で `sceneN.mp3` を各シーンに重ねる（`voiceover/README.md` のタイム表参照）。クリップが音声より短ければ操作の合間に静止を挟む。
- 価格表示は現行の **Free / Pro（$4.99月・$49.99年）** に統一（Basicは廃止）。
- 誇大表現を避ける（「法令準拠を保証」等は使わない）。
