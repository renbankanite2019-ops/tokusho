# Screenshot Captions / Overlay Text — Tokusho

> Dùng cho App Store listing screenshots. Mỗi ảnh thêm 1 dải (band) màu + headline để trông chuyên nghiệp và kể chuyện ①②③. Copy chữ JA (chính) hoặc EN (nếu làm bản tiếng Anh).

## Style chung (áp dụng cho cả 3 ảnh — để đồng bộ)
- **Ảnh nền:** screenshot 1600×900 (đã chụp).
- **Dải caption:** thanh ngang ở **trên cùng**, full-width, cao ~140px, nền màu **`#008060`** (xanh, khớp icon).
- **Headline:** chữ **trắng**, **bold (太字)**, cỡ ~**52px**, căn giữa hoặc căn trái cách lề 48px.
- **Subtext (nếu có):** chữ trắng/xám nhạt, ~**26px**, ngay dưới headline.
- **Badge số:** ① ② ③ đặt đầu headline (tùy chọn), cùng màu trắng.
- **Font:** sans-serif gọn (Noto Sans JP / Hiragino / hệ thống).
- Giữ **cùng vị trí, cùng cỡ chữ, cùng màu** trên cả 3 ảnh.

## Cách thêm caption nhanh

### Canva (dễ nhất)
1. Tạo design **Custom size 1600×900**.
2. Upload screenshot → kéo làm nền (phủ kín).
3. Thêm **Rectangle** ở trên cùng, full-width, cao ~140px, màu `#008060`.
4. Thêm **Text** lên rectangle: dán Headline (bold ~52) + Subtext (~26), màu trắng.
5. **Share → Download → PNG** (giữ 1600×900).
6. Lặp lại cho 3 ảnh, **giữ y nguyên vị trí/cỡ chữ**.

### Photopea (miễn phí, không cần tài khoản)
1. File → New → 1600×900.
2. File → Open & Place → screenshot (phủ kín).
3. Vẽ hình chữ nhật trên cùng (màu #008060), thêm Text trắng (Headline + Subtext).
4. File → Export as → PNG.

## Lưu ý
- **Tránh** chữ kiểu "保証 / 完全対応 / 100%" trên ảnh (景表法). Các caption trên đã an toàn ("入力するだけ / プレビュー / 公開").
- Caption ngắn, dễ đọc; đừng che mất nội dung chính của screenshot.
- 3 ảnh cùng style → trang listing nhìn nhất quán, chuyên nghiệp.
