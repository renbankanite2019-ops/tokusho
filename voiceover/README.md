## 再生成・調整
```
python voiceover/generate_voiceover.py
```
- **声を変える**：`generate_voiceover.py` の `VOICE` を変更
  - 女性: `ja-JP-NanamiNeural`（既定） / 男性: `ja-JP-KeitaNeural`
- **話速**：`RATE`（例 `"-8%"` でさらにゆっくり、`"+0%"` で標準）
- 利用可能な日本語音声の一覧: `edge-tts --list-voices` で `ja-JP` を確認
